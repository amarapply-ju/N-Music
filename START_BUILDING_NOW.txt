🔧 TROUBLESHOOTING

Build fails with "ANDROID_HOME":
  export ANDROID_HOME=$HOME/Android
  Then try: ./build-nmusic.sh

Build fails with "JAVA_HOME":
  export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64
  Then try: ./build-nmusic.sh

No APK file after build:
  Check: ls -la app/build/outputs/apk/mobile/debug/
  If empty, re-run: ./build-nmusic.sh

App crashes on launch:
  Check logs: adb logcat | grep -i error
  Try: adb shell pm clear moe.koiverse.archivetune
  Then reopen app

Font/theme not showing:
  Clear app cache: adb shell pm clear moe.koiverse.archivetune
  Force stop and restart app

More help:
  Read: /home/jeopardous4/Downloads/ArchiveTune-13.2.0/N-MUSIC_READY_TO_BUILD.txt
